npm install tailwind-rn     // https://github.com/vadimdemedes/tailwind-rn

npx setup-tailwind-rn

npm i twrnc                 // https://www.npmjs.com/package/twrnc


**You NEED TWO Terminals***

Terminal A = npm run dev:tailwind   // To watch for changes
    OR       npm run build:tailwind // to just build

Terminal B = expo start
